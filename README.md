# XTSAES-CIS

## Onboarding and Running Application

1. Clone

   ```
   git clone https://github.com/farhanfarasdak/XTSAES-CIS.git
   cd XTSAES-CIS
   cd src
   ```

2. Compile

   ```
   javac *.java
   ```

3. Run Application

   ```
   java Main
   ```
